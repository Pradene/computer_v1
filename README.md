# Computer_v1

This prokect is a Rust-based application designed to solve polynomial equations of 2nd or less degree. The project takes polynomial equations as input and computes the roots of the equation using various numerical methods. It's an efficient tool for solving polynomial equations in mathematical, engineering, and scientific applications.

## Installation
To use the project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/Pradene/computer_v1.git
    ```

2. Run the project:

    ```bash
    cargo run "2x^2 - 3x + 4 = 0"
    ```
